Scam

James B. Moen
January 17, 2014

The list of scams is as long as it is wide and new ones spring up every day to meet consumer needs.

[MAD 1995]

This directory contains source files for Scam, an interpreter for a toy language that is similar to a subset of Scheme [SPE 2007]. It is written in Orson [MOE 2014], and is intended to run under Unix-like systems, such as GNU/Linux. Scam was written to test the Orson library’s garbage collector, signal handler, and Unicode character methods. These files contain Orson source code for the Scam interpreter. The allegedly interesting ones are marked with ‘✗’.
